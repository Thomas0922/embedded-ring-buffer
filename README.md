## 專案簡介

Ring Buffer（環形緩衝區）是嵌入式系統中最常用的資料結構之一，特別適合處理串流數據和生產者-消費者問題。本專案提供了一個高效、可靠的實作，具備以下特點：

- **零動態記憶體分配**：所有記憶體在初始化時分配
  **固定時間複雜度**：所有操作皆為 O(1)
- **ISR安全**：適用於中斷服務程序
- **記憶體高效**：最小化結構體overhead
  **API簡潔**：易於整合到現有專案

## 核心特性

- 批次讀寫API

- 可用空間查詢

- [ ] 多執行緒安全版本
